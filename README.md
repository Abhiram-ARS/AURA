# AURA – ADB Utility for Remote Administration

AURA is a Python-based GUI tool that simplifies Android Debug Bridge (ADB) operations through an easy-to-use desktop interface. It allows users to pair, connect, manage, and monitor Android devices wirelessly without relying completely on terminal commands.

Built using Python and CustomTkinter, AURA provides a modern graphical interface for executing common ADB tasks efficiently on both Windows and Linux systems.

---


# Technologies Used

* Python 3
* CustomTkinter
* ADB (Android Debug Bridge)
* Threading Module
* Subprocess Module

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/AURA.git
cd AURA
```

### Install Requirements

```bash
pip3 install -r requirements.txt
```

```bash
sudo apt install adb
```

### Running the Application

```bash
python3 main.py
```

# How to Use

## Pair Device

1. Enable Wireless Debugging on your Android device
2. Open Pairing in AURA
3. Enter:

   * IP Address
   * Pairing Port
   * Pairing Code
